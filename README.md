## Propósito
El objetivo de este proyecto es simular y comparar el comportamiento de diferentes estructuras de datos tipo árbol (BST, Splay Tree y Red-Black Tree) en la gestión de procesos de un sistema operativo. Se va a analizar lo que es su eficiencia en búsqueda, inserción y número de iteraciones utilizando el concepto de vruntime.

## Estructura del proyecto
- structure.py: implementaciones de todas las clases necesarias 
- main.py: archivo principal para ejecutar las simulaciones
- docs/: contiene el diagrama UML

## Ejecución
1.Instalar dependencias:
pip install graphviz
descargar el programa de graphviz

2.Ejecutar el programa:
python main.py
